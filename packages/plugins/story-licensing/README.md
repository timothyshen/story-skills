# story-licensing

AI-powered assistance for Story Protocol licensing, PIL terms configuration, and derivative IP registration.

## Skills

### licensing

Guides developers through licensing IP on Story Protocol, covering:

- Programmable IP License (PIL) terms configuration
- PILFlavor presets (Non-Commercial Social Remixing, Commercial Use, Commercial Remix, Creative Commons Attribution)
- Attaching license terms to IP Assets
- Minting license tokens
- Registering derivative IP (with and without license tokens)
- commercialRevShare encoding (percent vs contract uint32)
- Contract addresses for Aeneid testnet and mainnet

## Installation

```bash
npx skills add storyprotocol/story-skills
```

## Usage

The skill activates contextually when you mention:

- "license", "PIL", "license terms", "PILFlavor"
- "derivative", "remix", "commercial use"
- "attach license", "commercialRevShare"
